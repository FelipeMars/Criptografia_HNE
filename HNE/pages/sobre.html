CRIPTOGRAFIA HNE-20

HEIMDALL NUNTIUS ENCRYPTION 2020

A criptografia Heimdall Nuntius Encryption 2020 (HNE-20) foi desenvolvida em outubro de 2020 pelo grupo de estudante da Universidade Paulista Alex, Felipe, Saymon e Victor.

Atualmente apenas mensagens de até 128 caracteres são possíveis de se criptografar utilizando chaves de até 64 caracteres. Ela conta com um sistema de chave por arquivo de extensão, tanto antigas como modernas, podem ser quebradas com determinado tempo e investimento utilizando técnicas como força bruta, porém até mesmo com uma técnica dessas tomaria um grande tempo já que existiriam um enorme número de possibilidades, requisitando um grande poder de processamento.

A criptografia funciona solicitando ao usuário uma mensagem e uma chave, a chave é convertida para um número de acordo com a posição na lista de caracteres, após estar com a chave final armazenada em uma variável, será feito a criptografia da mensagem de letra em letra, para cada letra é somado de número em número presente na variável, o resultado da soma irá equivaler a uma letra no index que logo será exibida para o usuário. Dessa forma, a criptografia nunca permitirá que um caractere seja equivalente a outro em todas as situações, um único caractere pode equivaler a ele mesmo e a outros 9 caracteres diferentes, entre esses caracteres é possível se transformar em números, pontuações ou símbolos.

Para a descriptografia é necessário que o usuário entre com a mensagem criptografada e a chava, se a chave coincidir com a que foi passada durante a criptografia, será feito o reverso da operação para cifrar a mensagem, informando assim a mensagem decifrada, caso não coincidir com a chave proposta na criptografia, então o resultado será uma mensagem totalmente diferente da original.

LISTA DE CARACTERES ACEITOS

+ : . , - “ / = * # @ $ ! % ( ) ?


O uso de qualquer caractere não presenta na lista resultará em falha na criptografia ou descriptografia. Somente letras minúsculas são permitidas.


Essa tabela foi embaralhada usando uma técnica de anagramas, dessa forma existem cerca de 2,521728 x 10117^10 (número convertido para notação cientifica) de possibilidades de montar essa lista de caracteres o que irá exigir um longo período de processamento para replicar corretamente essa lista.

SITE DA CRIPTOGRAFIA

Inicialmente a criptografia foi desenvolvida em linguagem de programação Python sem interface (rodando através de console), com isso surgiu a idaia de disponibilizar a criptografia para qualquer um com acesso a internet, para isso foi preciso converter de Python para JS. O site foi construido através de HTML, CSS e JavaScript por Felipe Martins. Atualmente o site conta com 4 páginas:

Página para criptografar e descriptografar mensagens utilizando uma chave.

Uma página que possibilita iniciar um chat privado utilizando uma chave para criptografar as mensagens. (Em breve).

Uma página para enteder sobre o funcionamento e história da criptografia.

E uma página para entrar em contato com o desenvolvedor.
